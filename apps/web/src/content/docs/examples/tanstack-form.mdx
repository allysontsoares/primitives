# TanStack Form + Zod

Learn how to integrate Kairo with TanStack Form and Zod to construct performant, type-safe date picker forms.

## Key Features
* **Granular Reactivity**: Standard TanStack Form subscription model.
* **Inline Schema Validation**: Validate inputs inline using Zod schema helper.
* **Agnostic Primitives**: Seamless field state bindings to Kairo elements.

## Code Example

Below is a complete, production-ready implementation of a date picker using TanStack Form:

```tsx
"use client";

import { useForm } from "@tanstack/react-form";
import { DatePicker } from "@at5/kairo";
import * as z from "zod";

const dateSchema = z.date({
  required_error: "Please select a date.",
}).refine((d) => d.getDay() !== 0 && d.getDay() !== 6, {
  message: "Appointments are only available on weekdays.",
});

export function TanstackAppointmentForm() {
  const form = useForm({
    defaultValues: {
      date: undefined as Date | undefined,
    },
    onSubmit: async ({ value }) => {
      console.log("Form values submitted:", value);
    },
  });

  return (
    <form
      onSubmit={(e) => {
        e.preventDefault();
        e.stopPropagation();
        form.handleSubmit();
      }}
      className="space-y-4 max-w-sm"
    >
      <form.Field
        name="date"
        validators={{
          onChange: dateSchema,
        }}
      >
        {(field) => (
          <div className="flex flex-col gap-1.5 relative">
            <label htmlFor={field.name} className="text-sm font-medium text-zinc-300">
              Select Weekday
            </label>

            <DatePicker.Root
              mode="single"
              value={field.state.value}
              onValueChange={(val) => {
                field.handleChange(val);
              }}
            >
              <div className="flex items-center gap-2 rounded-xl border border-zinc-900 bg-zinc-950 px-4 py-2.5 text-sm focus-within:ring-1 focus-within:ring-indigo-500 focus-within:border-indigo-500 transition-all">
                <DatePicker.Input
                  id={field.name}
                  className="flex-1 bg-transparent text-zinc-100 outline-none font-mono text-[12px]"
                  aria-invalid={field.state.meta.errors.length > 0 ? true : undefined}
                />
                <DatePicker.Trigger className="text-zinc-500 hover:text-zinc-100">
                  📅
                </DatePicker.Trigger>
              </div>

              <DatePicker.Content portal={true} side="bottom" align="start" sideOffset={6} className="w-[280px] rounded-xl border border-zinc-900 bg-black p-3.5 shadow-2xl backdrop-blur-md">
                <DatePicker.ViewControl className="flex items-center justify-between mb-2">
                  <DatePicker.PrevTrigger className="h-7 w-7 flex items-center justify-center rounded-lg bg-transparent text-zinc-500 hover:bg-zinc-900 hover:text-white" />
                  <DatePicker.ViewTrigger className="text-xs font-semibold text-zinc-350 hover:bg-zinc-900 transition-colors font-display" />
                  <DatePicker.NextTrigger className="h-7 w-7 flex items-center justify-center rounded-lg bg-transparent text-zinc-500 hover:bg-zinc-900 hover:text-white" />
                </DatePicker.ViewControl>

                <DatePicker.View view="day">
                  <DatePicker.Grid header={<DatePicker.WeekDays className="flex w-full mb-1.5 text-zinc-650 font-bold text-[10px] uppercase font-display" />}>
                    {({ weeks }) => (
                      <tbody className="flex flex-col gap-1">
                        {weeks.map((week, wi) => (
                          <tr key={wi} className="flex w-full justify-between mt-0.5">
                            {week.map((day, di) => (
                              <DatePicker.Day
                                key={di}
                                date={day}
                                className="h-8 w-8 flex items-center justify-center rounded-lg text-xs font-mono font-medium text-zinc-350 hover:bg-zinc-900 hover:text-white aria-selected:bg-indigo-600 aria-selected:text-white outline-none"
                              />
                            ))}
                          </tr>
                        ))}
                      </tbody>
                    )}
                  </DatePicker.Grid>
                </DatePicker.View>
              </DatePicker.Content>
            </DatePicker.Root>

            {field.state.meta.errors.length > 0 && (
              <p role="alert" className="text-xs text-red-400 mt-1">
                {field.state.meta.errors.join(", ")}
              </p>
            )}
          </div>
        )}
      </form.Field>

      <button
        type="submit"
        className="rounded-lg bg-blue-600 px-4 py-2 text-sm font-semibold text-white hover:bg-blue-500 transition-colors"
      >
        Submit
      </button>
    </form>
  );
}
```
